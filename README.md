# Auth Microservice
En este repositorio se encuentran dos proyectos con Python y FastAPI para la creación de un microservicio de autenticación de alta concurrencia.

---

El proposito de este repositorio es la creación de dos microservicios basicos para gestionar la autenticación de usuarios. Estos microservicios se comunican mediante llamadas HTTP. Además, el microservicio de autenticación provee un mecanismo de autenticación para funcionar como Indentity Provider (IdP). Cada microservicio cuenta con su propia base de datos para separar la responsabilidad y dependencia entre estos.
